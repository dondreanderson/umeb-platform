UMEB Nonprofit Management Platform


Create a comprehensive nonprofit management platform for United Men Empowering Brotherhood (UMEB) organizations.

CORE FEATURES:
1. Donor Management - profiles, donation history, tax receipts
2. Payment Processing - Stripe integration for donations
3. Volunteer Management - registration, scheduling, hour tracking
4. Event Management - calendar, registration, RSVP, check-in
5. Member Election & Voting - nominations, secure voting, results dashboard
6. Grant Management - tracking, deadlines, reporting
7. Financial Dashboard - analytics, budgets, reports
8. Communication Suite - email campaigns via SendGrid
9. Member Portal - OAuth2 auth, personal dashboard
10. Admin Dashboard - user management, reports, audit logs

TECHNICAL STACK:
- Frontend: React 18 with Next.js 14, TypeScript, Tailwind CSS, WCAG 2.2 AA
- Backend: Python FastAPI with Pydantic, JWT auth, RBAC
- Database: PostgreSQL 15 with migrations
- Integrations: Stripe, SendGrid, AWS S3
- Infrastructure: Docker, Terraform, Kubernetes
- Deployment: AWS ECS, RDS, CloudFront
